# 投标材料素材生成快速指南

## ✨ 新功能：一键生成投标材料

### 🎯 功能特点

我们为您添加了醒目的快速生成入口，让您能够更便捷地生成投标材料！

### 📋 三种生成模式

#### 1. 📄 生成商务材料
- **用途**：生成商务资质、项目经验、服务承诺等商务部分材料
- **适用场景**：需要快速准备商务标书内容

#### 2. 🔧 生成技术材料  
- **用途**：生成技术方案、系统架构、实施计划等技术部分材料
- **适用场景**：需要快速准备技术标书内容

#### 3. 🎯 批量生成所有材料
- **用途**：一次性生成商务和技术所有材料
- **适用场景**：需要快速准备完整标书

### 🚀 使用步骤

#### 方式一：快速生成模式（推荐）

1. **点击快速按钮**
   - 在主界面找到紫色的"投标材料素材生成"卡片
   - 点击对应的生成按钮（商务/技术/批量）

2. **上传招标文件**
   - 选择"上传招标文件"
   - 支持格式：PDF、Word、PPT

3. **填写生成要求（可选）**
   - 输入具体的材料生成要求
   - 例如："重点阐述公司资质和项目经验"
   - 如果不填，系统会根据招标文件自动分析

4. **选择使用知识库（可选）**
   - 勾选"使用知识库"以检索历史资料
   - 提高材料质量和相关性

5. **点击开始生成**
   - 点击"🚀 开始生成"按钮
   - 等待系统生成材料
   - 查看生成结果

6. **下载生成的材料**
   - 点击"下载材料"按钮
   - 保存为文本文件

#### 方式二：批量生成模式

1. 点击"🎯 批量生成所有材料"按钮

2. 上传招标文件

3. （可选）填写生成要求和选择知识库

4. 点击"🚀 开始生成"

5. 查看进度条，等待所有材料生成完成

6. 分别下载商务材料和技术材料

#### 方式三：高级配置模式

如果需要更详细的配置，可以：

1. 在侧边栏切换到"✍️ 投标材料生成"模式

2. 配置详细的参数：
   - 招标文件
   - 材料类型
   - 知识库路径
   - 生成要求

3. 点击"✍️ 开始生成材料"

### 💡 使用技巧

#### 提高生成质量的技巧

1. **填写详细的生成要求**
   ```
   例如：
   - 生成商务材料，重点突出：
     - 公司资质（ISO认证、高新技术企业等）
     - 类似项目经验（政府、金融、教育行业）
     - 服务优势（7*24小时支持、定期巡检）
   ```

2. **使用知识库**
   - 将历史投标文件放入 `assets/knowledge_base` 目录
   - 点击侧边栏的"🔄 刷新索引"
   - 系统会自动检索相关历史资料

3. **批量生成后手动优化**
   - 先批量生成框架
   - 再根据具体项目特点进行个性化调整

### 📊 界面说明

- **位置**：主界面顶部，紫色渐变背景
- **功能**：三种快速生成按钮
- **特点**：醒目、易用、快速

- **触发条件**：点击任意快速生成按钮后进入
- **显示内容**：
  - 招标文件上传
  - 生成要求输入
  - 知识库选项

- **显示内容**：
  - 进度条显示生成进度
  - 状态文本显示当前任务
  - 生成完成后显示可展开的结果
  - 每个结果都有独立的下载按钮

### ⚠️ 注意事项

1. **文件格式要求**
   - 支持格式：PDF、Word（.docx, .doc）、PPT（.pptx, .ppt）
   - 文件大小：建议不超过100MB
   - 文件内容：确保招标文件完整可读

2. **生成要求**
   - 虽然生成要求是选填，但填写后能获得更精准的结果
   - 建议明确说明需要生成的内容重点
   - 可以包含行业关键词和项目特点

3. **知识库使用**
   - 首次使用需要点击"🔄 刷新索引"
   - 建议将相关历史文档放入知识库
   - 知识库路径可以在侧边栏配置

4. **生成时间**
   - 单个材料生成：约2-5分钟
   - 批量生成：约5-10分钟
   - 生成过程中请勿关闭浏览器
